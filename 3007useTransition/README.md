# useTransition

**Файлів, що використовують `useTransition`:** 0

**Версія React, в якій було вперше представлено хук:** 18.0

**Interface:** `const [startTransition, isPending] = useTransition(config)`

`useTransition` is a React Hook that lets you render a part of the UI in the background.

Це такий хук, що допоможе користувачу не сплатити 2 рази за той самий товар. Він нам дасть змогу компонент поставити в стан hold, поки відбувається якась дія.
